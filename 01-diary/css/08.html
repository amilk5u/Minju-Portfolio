문자열 속성 선택자
선택자 형태 설명
선택자[속성~=값] ex)ko kr 속성 안의 값이 특정 값을 단어로 포합하는 태그를 선택
선택자[속성|=값] ex)ko-kr 속성 안의 값이 특정 값을 단어로 포합하는 태그를 선택
선택자[속성^=값] 속성 안의 값이 특정 값으로 시작하는 태그를 선택
선택자[속성$=값] 속성 안의 값이 특정 값으로 끝나는 태그를 선택
선택자[속성*=값] 속성 안의 값이 특정 값으로 포함하는 태그를 선택
