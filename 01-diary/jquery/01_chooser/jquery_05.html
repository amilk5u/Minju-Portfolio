기본필터

:animated
$('div.aimated')
애니메이션 효과가 실행 중인 요소
:first
$('p:first')
box:first 모든 선상의 것들중 box 최초 첫번째를 선택한다
:last
$('p:last')
모든 선상의 최종마지막의 p를 선택한다
:even
$('tr:even')
인덱스가 짝수인 요소
:odd
$('tr:odd')
인덱스가 홀수인 요소
:eq(index)
$('ul li:eq(3)')
인덱스(배열)값은 0 부터 그러므로 리스트는 4번째이다
:gt(index)
$('ul li:gt(3)')
선택한 요소보다 인덱스가 큰 요소
:not(selector)
$('input:not(.intro)')
지정한 선택자와 일치하지 않는 요소(인풋에서 인트로로 클래스가 없는 요소를 선택)
:header
$(':header')
h1,h2,h3,h4,h5,h6 제목태그를 모두선택
:lt(index)
$('u li:lt(3)')
선택한 요소보다 인덱스가 작은 요소
:root
$('root')
웹 페이지 최상위(루트) 요소(html 요소)
:lang(language)
$('p:lang(ko)')
lang 속성값이 ko인 요소 ex) < p lang="lt">

내용필터

:contains(text)
$('p:contains("hello")')

$(':empty')

:parent
$('td:parent')
자식 노드를 갖고 있는 요소 (트리구조나무를 거꾸로했을때 체계적으로 구조화한것)

속성

[attribute]
#('[href]')
속성
[attribute=value]
$('[href="default.html"]')
다음과 같은 속성과 값을 가진 요소
[attribute!=value]
$('[href!="a.html"]')
다음과 같은 속성의 값이 아닌 요소
[attribute=value]
$('[title|="Tomorrow"]')
속성명의 값이 일치하는 요소 혹은 value값과 뒤이은 하이픈부터 시작하는 값과 일치하는 요소
[attribute$=value]
$('[href*=".jpg"]')
속성명의 값이 value값으로 끝나는 요소
[attribute^=value]
$('[title^="Tom"]')
속성명의 값이 value값으로 시작하는 요소
[attribute~=value]
$('[title~="hello"]')
속성명의 값에서 공백으로 구분된 목록 중 하나가 value 값인 요소
[attribute*=value]
$('[title*="hello"]')
속성명 값이 value값 전체를 포함하는 요소
