전후 문자 선택자

전후 문자 선택자는 특정 태그의 전후에 위치하는 공간을 선택하는 선택자이다

선택자 형태 설명
::after 태그 뒤에 위치하는 공간을 선택
::before 태그 앞에 위치하는 공간을 선택

전후 문자 선택자에는 centent속성을 사용할 수 있다
